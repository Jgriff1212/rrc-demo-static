# Assets Directory

This directory contains app assets including icons, splash screens, and sample art packs.

## Required Assets

### App Icons
- `icon.png` - 1024x1024px app icon
- `adaptive-icon.png` - 1024x1024px Android adaptive icon (foreground)
- `favicon.png` - 48x48px web favicon

### Splash Screen
- `splash.png` - 2048x2732px splash screen

### Sample Art Packs

Three sample art pack styles are included for demonstration:

1. **Starter Pack** - Default style, abstract geometric patterns
2. **Vibrant Pack** - Bright, colorful illustrations
3. **Minimal Pack** - Clean, minimalist designs

These are placeholder images. In production, integrate with actual AI image generation API.

## Generating Icons

Use the following tools to generate app icons from a single source:
- [expo-app-icon-generator](https://www.npmjs.com/package/@expo/app-icon-generator)
- [expo-splash-screen](https://docs.expo.dev/guides/splash-screens/)

## Placeholder Images

During development, placeholder images from [Lorem Picsum](https://picsum.photos/) are used.
Replace with actual AI-generated images in production.
